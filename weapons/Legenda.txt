The weapons file are organised as follows:
1) NUMBER OF THE COMMAND RELATED TO THE WEAPON
2) NUMBER OF DAMAGE TOKEN THE WEAPON GIVES
3) NUMBER OF MARKS THE WEAPON GIVES
4) TELEPORTER FLAG char
5) TELEPORTER FLAG for the target
6) NUMBER OF RED AMMOCUBES COST
7) NUMBER OF BLUE AMMOCUBES COST
8) NUMBER OF YELLOW AMMOCUBES COST
9) FLAG THAT IDENTIFIES IF THE EFFECT HAS A RELATED EFFECT DIFFERENT FROM THE ORIGINAL
